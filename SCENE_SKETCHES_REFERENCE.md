# Scene Sketches Reference

All scene sketch suggestions used as fallback ideas in the card design step.
Each sketch has an **audience tag** that controls who sees it:

- **all** — shown to every age group
- **young** — child/teen only (filtered out for adults)
- **mature** — adult/senior only (filtered out for children)

Source: `src/lib/card-recipes.ts`
